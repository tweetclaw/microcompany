## 使用指南

### 启动应用

```bash
# 使用日志管理脚本启动
./dev.sh

# 或使用标准方式
npm run tauri dev
```

### 配置 API Key

**方法 1: 环境变量**
```bash
export ANTHROPIC_API_KEY='your-api-key-here'
./dev.sh
```

**方法 2: 配置文件**
创建 `~/.microcompany/config.json`:
```json
{
  "anthropic_api_key": "your-api-key-here",
  "model": "claude-sonnet-4-6"
}
```

详见：[API配置指南.md](API配置指南.md)
