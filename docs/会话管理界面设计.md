## 实现方案

### 1. 组件结构

**Sidebar.tsx** - 侧边栏组件
- 可折叠的侧边栏
- 显示会话列表（使用 SessionList 组件）
- 点击会话 → 提示用户需要重启应用切换会话

**ChatInterface.tsx** - 更新对话界面
- 添加侧边栏切换按钮（☰）
- 集成 Sidebar 组件
- 管理侧边栏展开/收起状态

### 2. 用户体验流程

#### 查看历史会话
1. 在对话界面点击左上角的 ☰ 按钮
2. 侧边栏从左侧滑出
3. 显示所有历史会话列表

#### 切换会话
1. 在侧边栏点击其他会话
2. 弹出提示："切换会话需要重启应用。是否继续？"
3. 用户确认 → 关闭应用（或保存会话路径到 localStorage）
4. 用户重新启动应用 → 自动加载选中的会话

#### 删除会话
1. 在侧边栏点击会话的删除按钮
2. 确认删除
3. 会话从列表中移除

### 3. 遵守工作目录铁律

由于一个进程只能对应一个工作目录，切换会话的实现方式：

**方案 A：提示重启**
- 点击其他会话 → 提示用户需要重启应用
- 将选中的会话路径保存到 localStorage
- 用户手动重启应用 → 自动加载该会话

**方案 B：自动重启（推荐）**
- 点击其他会话 → 确认对话框
- 保存选中的会话路径到 localStorage
- 调用 Tauri API 重启应用
- 应用启动时检查 localStorage，如果有保存的路径，直接加载
