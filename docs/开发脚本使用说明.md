# MicroCompany 开发脚本使用说明

## 启动开发服务器

使用 `dev.sh` 脚本启动开发服务器，所有日志会自动保存到 `dev.log` 文件：

```bash
./dev.sh
```

## 查看日志

### 实时查看日志
```bash
tail -f dev.log
```

### 查看最近的日志
```bash
tail -100 dev.log
```

### 搜索日志中的错误
```bash
grep -i error dev.log
```

## 脚本功能

- ✅ 自动清空旧日志
- ✅ 启动开发服务器
- ✅ 将所有输出（stdout 和 stderr）重定向到 `dev.log`
- ✅ 同时在终端显示输出（使用 `tee` 命令）

## 关于 mach port 错误

如果你在日志中看到这个错误：
```
error messaging the mach port for IMKCFRunLoopWakeUpReliable
```

这是 macOS 系统的已知警告，与输入法框架相关，**可以安全忽略**，不会影响应用功能。

## 其他启动方式

如果你不想使用日志脚本，仍然可以使用标准方式启动：

```bash
npm run tauri dev
```
