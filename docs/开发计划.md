## 项目概述

MicroCompany 是一个基于 Tauri 的 Claude Code GUI 应用，目标是为 Claurst 提供图形界面，同时保留完整的工具调用能力。

### 核心目标

1. **完整的工具调用能力** - AI 可以读取文件、执行命令、搜索代码
2. **工作目录上下文感知** - AI 完全了解当前工作目录的状态
3. **多轮对话支持** - 保持对话历史和上下文
4. **流式响应** - 实时显示 AI 回复
5. **美观的 UI** - 基于 logo 的深色主题设计

## 关键技术挑战

### 1. Claurst API 理解
**挑战**: Claurst 的 API 可能不够文档化  
**方案**: 
- 阅读 Claurst 源码
- 查看 Claurst CLI 的实现
- 从简单用例开始

### 2. 异步处理
**挑战**: Claurst 的 API 是异步的，需要正确处理  
**方案**:
- 使用 Tokio 作为异步运行时
- 正确使用 async/await
- 避免死锁（注意 Mutex 的使用）

### 3. 工具调用集成
**挑战**: 需要拦截和处理工具调用  
**方案**:
- 实现工具调用回调
- 在 Rust 侧执行工具
- 将结果返回给 Claurst

### 4. 流式响应
**挑战**: 需要实时显示 AI 回复  
**方案**:
- 使用 Tauri 事件系统
- 前端监听事件并更新 UI
- 处理流式中断
